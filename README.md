# Pathan Samaj Trust Portal

આ પઠાન સમાજ ટ્રસ્ટ માટે બનાવેલ વેબ એપ્લિકેશન છે.

## Features (સુવિધાઓ)

*   **સભ્ય રજિસ્ટ્રેશન:** નવા સભ્યોની નોંધણી (પટેલ, મુસ્લિમ, સ્વામિનારાયણ સમાજ).
*   **સભ્ય યાદી:** બધા સભ્યોનું લિસ્ટ અને સર્ચ ઓપ્શન.
*   **ફંડ મેનેજમેન્ટ:** ડોનેશન અને ફંડ જમા કરવાની સુવિધા.
*   **દીકરી યોજના:** દીકરીઓ માટે સહાય યોજના અને ઓટોમેટિક પાત્રતા ચેકિંગ.
*   **રસીદ:** ગુજરાતીમાં રકમ અને ઓટોમેટિક રસીદ જનરેશન.

## Technology Used

*   React.js
*   TypeScript
*   Tailwind CSS
*   Vite

## How to Run (કેવી રીતે ચલાવવું)

1.  `npm install`
2.  `npm run dev`
